# OPUS-web3.0
OPUS is a platform for job offers on the web 3.0

## Features

- Send mail
- Post a job
- View a post
- Report a job
- General View of posts
- Filtering posts by niches

## Tech Stack

**Client:** React, Redux, TailwindCSS, Canvas.

**Server:** Node, Express, Nodemailer, JavaScript, Ejs, Mysql.
